2. Dados que Coletamos

2.1 Fornecidos pelo usuário no cadastro e onboarding: Nome de exibição; data de nascimento; número de telefone, verificado por código SMS via Firebase Authentication; identidade de gênero e preferência de conexão; até 3 fotos; coordenadas de latitude e longitude; música favorita (título, artista, identificador Deezer e URL de capa); respostas ao chat do curador durante o onboarding; código de indicação, quando informado voluntariamente.

2.2 Gerados pelo sistema: Essências — 6 traços de personalidade com pesos numéricos, derivados por inteligência artificial com base nas respostas do onboarding; vetor de embedding semântico (1536 dimensões) gerado a partir das essências; score de compatibilidade (0–100) calculado no momento de cada match; textos de apresentação e perguntas gerados pela IA; histórico de matches diários e status de cada ciclo; preferências de busca e contador de dias sem match; token FCM do dispositivo para envio de notificações push; data e versão dos Termos de Uso aceitos pelo usuário.

2.3 Coletados automaticamente: Logs de acesso do servidor (horário, rota acessada, status HTTP e tamanho da resposta), sem registro de IP ou user-agent no padrão atual; token de acesso JWT e hash do refresh token.

3. Como Usamos os Dados

Verificação do número de telefone por SMS para autenticação segura; formação do par diário pelo algoritmo de matching; geração de textos personalizados pela IA; envio de notificações push (match formado, nova mensagem, encerramento do chat); rastreamento de indicações e distribuição de benefícios em campanhas ativas; registro de aceite dos Termos de Uso para fins legais; segurança e prevenção de fraudes; cumprimento de obrigações legais.

4. Compartilhamento com Terceiros

OpenAI (EUA): recebe dados do usuário para geração de textos e extração de essências. Os dados enviados incluem nome, idade, identidade, preferências, música favorita, histórico do chat de onboarding e essências com pesos. Consulte a política em openai.com/privacy.

OpenRouter (EUA): recebe mensagens do chat para processamento por modelos de linguagem em funcionalidades específicas do backend. Consulte a política em openrouter.ai.

Firebase / Google (EUA): utilizado para verificação do número de telefone por SMS (Firebase Authentication) e para entrega de notificações push (Firebase Cloud Messaging). Consulte a política em policies.google.com/privacy.

Amazon Web Services (EUA): o serviço Amazon Rekognition pode ser utilizado para verificação facial comparando foto de perfil e selfie, quando ativado. Consulte a política em aws.amazon.com/privacy.

Deezer (França): utilizado para busca e exibição de metadados de músicas. Consulte a política em deezer.com/legal/privacy.

Não vendemos dados pessoais a terceiros.

5. O que Outro Usuário Vê de Você

Quando você é pareado com outro usuário, ele tem acesso a: nome de exibição, idade calculada, distância aproximada em quilômetros, lista de essências (nomes dos traços, sem pesos), fotos (com desfoque progressivo durante as primeiras horas do chat) e música favorita. Seu número de telefone, coordenadas exatas e demais dados internos não são exibidos ao par em nenhum momento.

6. Sistema de Indicações

Caso o usuário informe um código de indicação durante o cadastro, esse código é armazenado para fins de rastreamento de indicações e distribuição de benefícios ao indicador. Nenhum dado pessoal do usuário é compartilhado com o indicador — apenas a confirmação de que o cadastro foi realizado com o código informado.

7. Armazenamento e Retenção

Os dados são armazenados em banco de dados PostgreSQL hospedado em servidor próprio. Fotos e arquivos de mídia são armazenados em disco no servidor. Os dados permanecem armazenados enquanto a conta estiver ativa. Mensagens do chat não possuem prazo automático de exclusão e permanecem vinculadas ao histórico do match enquanto a conta existir. Ao excluir a conta, todos os dados pessoais são removidos permanentemente do banco de dados, incluindo mensagens, essências, histórico de matches e tokens. Arquivos de mídia armazenados no servidor são removidos em até 30 dias após a exclusão da conta.

8. Seus Direitos (LGPD)

Nos termos da Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a: confirmar a existência de tratamento; acessar seus dados; corrigir dados incompletos ou desatualizados; solicitar anonimização, bloqueio ou eliminação; revogar consentimento; solicitar portabilidade; obter informações sobre compartilhamento. Para exercer seus direitos, entre em contato pelo e-mail linger.dating.app@gmail.com.

9. Segurança

Utilizamos verificação de número de telefone por SMS via Firebase Authentication no cadastro; tokens JWT com assinatura HMAC para autenticação nas requisições; refresh tokens armazenados apenas como hash SHA-256; armazenamento seguro de tokens no dispositivo (Android Encrypted Preferences / iOS Keychain); comunicação via HTTPS.

10. Menores de Idade

O Linger é destinado exclusivamente a maiores de 18 anos. Não coletamos intencionalmente dados de menores. Se identificarmos cadastro de menor, a conta será encerrada e os dados removidos.

11. Alterações

Atualizações relevantes a esta Política serão comunicadas pelo aplicativo com antecedência mínima de 15 dias.
